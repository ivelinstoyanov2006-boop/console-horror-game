# Console Horror Game (C++)

A text-based horror game built in C++ featuring interactive choices, branching outcomes, and atmospheric sound.

## About the Project
This project was created as a personal exploration of interactive storytelling and atmosphere in console-based applications.

## Features
- Decision-based gameplay
- Multiple possible endings
- Sound integration for immersion

## Technologies
- C++
- Console input/output

## How to Run

1. Ensure the `.wav` file is located in the same directory as the source code.
2. Open your terminal in that folder and compile the game by typing:
```bash
g++ House.cpp -o HouseGame
```
3. Run the generated executable by typing: `./HouseGame`
4. Follow the on-screen instructions.

Currently in development, with plans to expand into a full game using a game engine.
